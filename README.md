# netflyx

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Create config.json into src with this code:
```
{
    "API_KEY": "Your API key",
    "BASE_URL": "https://api.themoviedb.org/3/"
}
```
